HOW USING CONVERT HTML THEMES

INCLUDE CSS

First, include CSS files into your HTML head:

HTML
<!DOCTYPE html>
<html lang="en">
<head>
  <meta charset="UTF-8">
  <title>Document</title>
  <link type="text/css" rel="stylesheet" href="../convert-all-theme.css">
</head>
</html>

OPTIONS

If you don't like using all themes. You can using only theme.

<link type="text/css" rel="stylesheet" href="../convert-theme-default.css">
<link type="text/css" rel="stylesheet" href="../convert-theme-dark.css">

SCSS

You can @import file with Sass.

ALL
@import '../sass/convert-all-theme.scss';
@import '../sass/convert-theme-default.scss';
@import '../sass/convert-theme-dark.scss';
